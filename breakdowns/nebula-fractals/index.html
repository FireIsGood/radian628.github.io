Nebula Fractals Breakdown

This project contains what I like to call "nebula fractals." Many kinds of fractals may seem somewhat complicated or intricate, but are actually generated by a very simple underlying algorithm. Nebula fractals are no different.

To create a Nebula Fractal, start with a tree fractal. Play around with this generator to see how they work. Essentially, each branch has two branches branching off of it, each rotated a specific angle relative to the original branch, and scaled by a specific scale factor relative to the original branch.

To turn these tree fractals into nebula fractals, one has to do the following:

  1. Make the tree fractals 3D.
  2. Have the tree fractals branch off into three branches rather than two.
  3. Draw a point at the ends of the terminating branches, but don't actually draw any of the branches.
